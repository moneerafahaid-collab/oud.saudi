# منصة عَود السعودية

منصة رقمية لمزارعي حائل والقصيم — إعادة تدوير مخلفات النخيل والأشجار، ورش تدريبية، طلبات خدمة، ومساعد ذكي (عشيبة AI).

## معاينة التصميم

بعد تفعيل GitHub Pages، ستكون المعاينة متاحة على:

`https://<اسم-المستخدم>.github.io/<اسم-المستودع>/`

## التشغيل محلياً

```bash
npm install
npm run dev
```

## البناء

```bash
npm run build
```
